# Card 卡片

## Example

```jsx
<div style={{margin: '20px'}}>
  <Card style={{width: 200}}>
    Card component
  </Card>
</div>
```

## API

|   属性    |   类型   | 默认值 |   描述   |
|:---------:|:--------:|:------:|:--------:|
| className | `string` |   -    |   类名   |
|   style   | `object` |   -    | 行内样式 |
