# Icon 字体图标

## Example

```jsx
<Icon type='close' />
```

## API

|   属性    |     类型     | 默认值 |   描述   | 是否必填 |
|:---------:|:------------:|:------:|:--------:|:--------:|
| className |   `string`   |   -    |   类名   |    -     |
|   style   |   `object`   |   -    | 行内样式 |    -     |
|   type    |   `string`   |   -    |   类型   |    是    |
|  onClick  | `() => void` |   -    | 点击回调 |    -     |
