# Loading 加载中

## Example

```jsx
<Loading visible={visible} />
```

## API

|   属性    |   类型    | 默认值 |   描述   | 是否必选 |
|:---------:|:---------:|:------:|:--------:|:--------:|
| className | `string`  |   -    |   类名   |    -     |
|   style   | `object`  |   -    | 行内样式 |    -     |
|  visible  | `boolean` |   -    | 是否显示 |    -     |
|  size  | `number` `string` |   24    | 尺寸（即高度） |    -     |
|  color  | `string` |   `#537682`   | 颜色 |    -     |
