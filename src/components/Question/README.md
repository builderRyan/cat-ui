# Question

## Example

```jsx
<Question tips={<div>Question content</div>}  />
```

## API

|   属性    |       类型        | 默认值 |   描述   |
|:---------:|:-----------------:|:------:|:--------:|
| className |     `string`      |   -    |   类名   |
|   style   |     `object`      |   -    | 行内样式 |
|   tips    | `React.ReactNode` |   -    | 提示内容 |
